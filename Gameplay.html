Gameplay

All the modules of the Bomb Squad Game!

Welcome to the Bomb Squad! In this game, one player, the bomb defuser, is in charge of communicating the bomb setup to teammates, while the teammates have access to the bomb defusal manual try to figure out and tell their teammate how to defuse the bomb.

The bomb has a variety of different puzzle modules to solve; the players must solve the puzzle before the time runs out. Every module has a distinct set of instructions required to properly defuse it, but even the same module may have a different solution each time. Every mistake is tracked on the strike counter. Unlike bowling, you want to avoid strikes - each strike will speed up the countdown timer, and three strikes results in immediate combustion.

To begin the game, have your brave bomb volunteer insert a penny to activate the bomb defusal routine. Rounds are expected to take around 3-5 minutes, but may be longer or shorter depending on the trigger happiness of teammates.

Meet the Modules

Master Module

The Master Module contains both the countdown timer as well as the main PIC controller. It communicates with all the other modules to keep track of puzzle completion through the I2C protocol.

Switch Matrix Module

The Switch Matrix Module contains a variety of switches that have two states. The goal of the game is to communicate with teammates to figure out which state to set all the different switches in before locking in the final state.

Codeword Module

The Codeword Module presents the bomb defuser with five different letter slots, with each slot containing a unique pool of letters. The defuser needs to cycle through each letter slot and determine the available letters to form the codeword. The defuser's teammates have the list of possible codewords, and work with the defuser to figure out what the codeword could be based on the unique letter pools.

Combo Lock Module

The Combo Lock Module involves figuring out the combination numbers from teammates, and then entering in the numbers by turning the dial like a normal combination lock. If the player enters a wrong number, the module gives them a strike and resets.

Needy Module

The Needy Module is a simple module where the object is to remember to turn it off at every interval. If neglected, the bomb will explode.

Aux Cable Module

The Aux Cable Module presents a colorful assortment of Auxillary cables that will be set up in a random state. Based on the initial configuration, players will need to figure out the proper end-configuration by moving and unplugging cables.

Morse Code Module

The Morse Code Module involves finding out and entering a specific frequency on the slider. The module will constantly be blinking a morse code word. The Bomb Defuser must note the morse code letters and figure out the codeword, which corresponds to a specific frequency the teammates must communicate to the Defuser. This frequency will be entered and checked.
